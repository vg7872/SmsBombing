Git hub : 
1. To clone or download the project from github:
Git clone <URL>
ex.: git clone https://github.com/vg7872/StudentServiceApplication.git

2. Always, main branch gets downloaded.
So to checkout to any branch/feature branch, Please use this command :

git checkout <branchName>
ex.: git checkout feature/develop


3. To check the status of changes :
git status

4. Steps to push/upload the changes to github repository:

    a. git add *

    b. git commit -m "<Message>"
        ex..: git commit -m "Initial commit"

    c. git pull

    d. git push  || git push origin
